    О проекте

    Цели

    🗹 Создать базу данных, содержащую информацию о расположении инвентаря в кабинетах.

    🗹 Создать сайт, с помощью которого пользователь сможет работать БД.

    🗹 Ознакомиться с фреймворком Django, языком верстки HTML, фреймворком Bootstrap

    🗹 Оптимизировать командную работу, используя веб-сервис GitHub, основанный на системе контроля версий Git.

    Задачи

    Мы выполнили такие задачи:
    • Освоить технологии для реализации проекта
    • Создать базу данных
    • Сделать сайт с удобным интерфейсом
    • Сделать возможным редактирование объектов базы данных с сайта

    Что использовали для реализации проекта

    Git
    Используется для оптимизации работы команды. Может хранить старые версии проекта, что позволяет "откатиться" назад в случае неудачи.
    Django
    Позволяет упростить разработку благодаря встроенным шаблонам.
    HTML
    Применяется для создания видимой части сайтов, т.е. является основой Frontend'а проекта.

    Что изучили, пока делали проект

    Git
    Система управления и контроля версий.
    Django
    Бесплатный и свободный фреймворк для веб-приложений, написанный на Python.
    HTML
    Стандартный язык разметки документов во Всемирной паутине.
    CSS
    Формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
    JavaScript
    Прототипно-ориентированный сценарный язык программирования. Является диалектом языка ECMAScript.
    Bootstrap
    Фреймворк, включающий в себя набор инструментов для работы создания сайтов и приложений.
